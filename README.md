# Claude Skills

A personal collection of Claude skills for use with Cowork and Claude Code.

## Installation

Download the `.skill` file, open Cowork or Claude Code settings, navigate to Capabilities → Skills, and install from file.

## Building & Packaging

Skills are built using the Skill Creator in Cowork. To repackage a skill from source, open the skill creator (`/skill-creator` in Cowork) and point it at the skill folder.

## Skills

| Skill | Command | Description |
|-------|---------|-------------|
| grill | `/grill` | Interview relentlessly about any plan or decision, one question at a time |
| grill-dev | `/grill-dev` | Same as grill, but explores the codebase to answer questions where possible. Claude Code only. |
